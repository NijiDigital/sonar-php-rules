Versions of SSL/TLS protocol and cypher suites must be chosen carefully and should not be modified at runtime

Noncompliant Code Example

// SSLv3 is a deprecated version of SSL/TLS protocol
curl_setopt($curl, CURLOPT_SSLVERSION, CURL_SSLVERSION_SSLv3);
// Cypher suites should be chosen carefully
curl_setopt($curl, CURLOPT_SSL_CIPHER_LIST, "AES128-SHA:RC2-CBC-MD5:KRB5-RC4-MD5:KRB5-RC4-SHA:RC4-SHA:RC4-MD5")

  • MITRE CWE-327 - Use of a Broken or Risky Cryptographic Algorithm
  • OWASP Top 10 2017 Category A6 - Security Misconfiguration
