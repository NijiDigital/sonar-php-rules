The filter_input() function should not be used with FILTER_DEFAULT nor FILTER_UNSAFE_RAW filter option.

Noncompliant Code Example (Drupal 8)

    $search = filter_input(INPUT_GET, 'search', FILTER_UNSAFE_RAW);

See

  • MITRE CWE-20 - Improper Input Validation
  • OWASP Top 10 2017 Category A1 - Injection
