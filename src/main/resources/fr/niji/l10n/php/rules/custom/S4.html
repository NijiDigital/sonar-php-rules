Namespace import mechanism through the keyword use should be preferred over require and include functions. Using those functions may lead to local file inclusion attacks

Noncompliant Code Example

require_once('./config.php');

Exceptions

autoload.php ScriptHandler.php

See

  • MITRE CWE-97 - Improper Neutralization of Server-Side Includes (SSI) Within a Web Page
  • OWASP Top 10 2017 Category A5 - Broken Access Control
