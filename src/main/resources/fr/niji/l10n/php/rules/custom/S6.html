A dynamic database query should always have a constant string for its conditional or snippet arguments. Dynamically computed query snippets or condition strings may lead to SQL injection attacks

Noncompliant Code Example

$query->condition('w.severity', '5', $operator);

See

  • MITRE CWE-89 - Improper Neutralization of Special Elements used in an SQL Command ('SQL Injection')
  • OWASP Top 10 2017 Category A1 - Injection
