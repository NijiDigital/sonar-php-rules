If not property used, functions like system, shell_exec or `` may lead to arbitrary system commands execution.

There are some exceptions where those functions are allowed. They may be used in Drush commands, which are use for administrative tasks

Noncompliant Code Example

shell_exec("rm -rf /tmp/*");

See

  • MITRE CWE-78 - Improper Neutralization of Special Elements used in an OS Command ('OS Command Injection')
  • OWASP Top 10 2017 Category A1 - Injection
