Superglobal variables that contains user input should not be used directly. Prefer using the filter_input() function.

Noncompliant Code Example (Drupal 8)

    $value = $_POST['param1'];
    $query_string = "/var/www/html/images/".$_POST['image_name']."png";

See

  • MITRE CWE-20 - Improper Input Validation
  • OWASP Top 10 2017 Category A1 - Injection
