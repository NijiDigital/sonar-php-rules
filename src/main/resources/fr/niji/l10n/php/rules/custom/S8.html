In the Form API, using data from $form_state['input'] (Drupal 7) or $form_state->getUserInput()(Drupal 8) are a security risk.

Use data from $form_state['values'] (Drupal 7) or $form_state->values (Drupal 8) instead.

Noncompliant Code Example (Drupal 8)

class MyForm extends FormBase {
    public function validateForm(array &$form, FormStateInterface $form_state) {

        $email = $form_state->getUserInput('email');
    }}
